## Project Description
This project is the Week 8 assignment for React Fundamentals. It builds the base frontend for CommunityHub and demonstrates core React concepts: JSX, reusable components, props, state, event handling, controlled forms, list rendering, conditional rendering, and lifting state up.

## Technologies Used
- React
- Vite
- JavaScript (ES6+)
- CSS3
- npm

## Features
- CommunityHub layout with header, sidebar, main feed, profile section, and footer
- Reusable shared components (`Button`, `Card`, `Input`)
- Post feed with create, like, delete, and search/filter interactions
- Lesson 15 exercises included in the app
- Lesson 16 exercises included in the app
- Responsive layout for desktop and mobile

## How to Run
1. Clone the repository.
2. Open a terminal in the project root.
3. Install dependencies:
	`npm install`
4. Start the development server:
	`npm run dev`
5. Open the local URL shown in the terminal (usually `http://localhost:5173`).

## Lessons Learned
- React components are easier to maintain when each has one clear responsibility.
- Props help keep components reusable and consistent.
- State updates should be immutable when working with arrays and objects.
- Controlled forms make input handling and validation predictable.
- Lifting state up is essential when multiple components need shared data.

## Challenges Faced
- Integrating previously separate exercise components into one app flow.
- Making sure newly created posts matched the shape used by post cards.
- Balancing clarity of layout with reusable styling for all components.
